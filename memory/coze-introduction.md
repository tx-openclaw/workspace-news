# 扣子（Coze）编程平台介绍

## 一、扣子是什么？

**扣子（Coze）** 是字节跳动推出的 **AI Agent 智能办公平台**，致力于用 AI 重塑生产力与工作效率。它是一个低代码/无代码的 AI 应用开发平台，让用户无需编程基础就能快速创建和部署 AI 智能体（Bot）。

- **国际版**：https://www.coze.com
- **国内版**：https://www.coze.cn

## 二、核心功能

### 1. 智能体（Bot）创建
- **可视化编排**：通过拖拽组件、配置参数快速搭建 AI 助手
- **零代码开发**：无需编程经验，通过自然语言描述即可创建
- **模板丰富**：提供大量预置模板，涵盖写作、分析、客服等场景

### 2. 插件系统
- **内置插件**：搜索引擎、图片生成、文档处理、天气查询等
- **自定义插件**：支持接入外部 API，扩展 Bot 能力
- **插件市场**：社区共享的插件生态

### 3. 知识库
- **文档上传**：支持 PDF、Word、Markdown 等格式
- **知识检索**：Bot 可基于上传的知识库内容回答问题
- **实时更新**：知识库内容可随时更新维护

### 4. 工作流（Workflow）
- **流程编排**：可视化设计复杂的任务流程
- **条件分支**：支持 if-else 逻辑判断
- **循环处理**：批量数据处理能力

### 5. 多渠道发布
- **飞书/Lark**：一键发布到飞书机器人
- **微信公众号**：接入微信生态
- **API 调用**：提供标准 API 接口
- **Web 嵌入**：嵌入网站或应用

## 五、如何开始使用

### 步骤 1：注册账号
访问 https://www.coze.cn（国内版）或 https://www.coze.com（国际版）注册账号

### 步骤 2：创建 Bot
1. 点击"创建 Bot"
2. 输入 Bot 名称和描述
3. 选择基础模型

### 步骤 3：配置能力
1. 编写提示词（Prompt）
2. 添加插件
3. 上传知识库
4. 设计工作流

### 步骤 4：测试与发布
1. 在线测试 Bot 效果
2. 调优参数
3. 发布到目标渠道
